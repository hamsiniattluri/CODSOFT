# Tic-Tac-Toe AI

## Description
A Python-based Tic-Tac-Toe game where a human competes against an AI opponent. The AI uses the Minimax Algorithm to make optimal decisions and cannot be defeated.

## Features
- Human vs AI gameplay
- Minimax Algorithm
- Unbeatable AI
- Win and Draw detection
- Interactive console interface

## Technologies Used
- Python
- Minimax Algorithm
- Game Theory

## How to Run

python tic-tac-toe.py
